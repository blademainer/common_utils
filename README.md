common_utils
============
公共帮助类，工具包括文件工具、字符串工具、文件工具、注解工具，公共平台包括百度ip查询，其他的暂未加入。
maven依赖配置：
```xml
<dependency>
    <groupId>com.xiongyingqi</groupId>
    <artifactId>common_utils</artifactId>
    <version>0.0.7</version>
</dependency>
```
